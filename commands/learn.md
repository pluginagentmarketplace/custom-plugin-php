# /learn

Start your learning journey with the Developer Roadmap Plugin.

## Overview

This command guides you through:
- Choosing your career path
- Understanding prerequisite skills
- Creating a learning plan
- Getting started with resources

## Usage

Simply type `/learn` to begin an interactive learning path selection.

## What You'll Learn

Based on your chosen role, you'll get:

1. **Roadmap Overview**
   - Complete learning path
   - Key milestones and checkpoints
   - Time estimates

2. **Skill Breakdown**
   - Essential skills (must-have)
   - Important skills (recommended)
   - Advanced topics (optional)

3. **Learning Resources**
   - Free and paid options
   - Video tutorials
   - Interactive practices
   - Books and documentation
   - Community forums

4. **Projects & Practice**
   - Real-world projects
   - Difficulty progression
   - Portfolio builders
   - Hands-on exercises

## Available Career Paths

### Tech Development (9 Agents)
- **Web & Frontend** - React, Vue, Angular, modern web
- **Backend & Servers** - Node.js, Python, Java, Go, Rust
- **Mobile & Cross-Platform** - iOS, Android, Flutter, React Native
- **Data & Analytics** - Data Engineering, Analytics, Science
- **AI & Machine Learning** - LLMs, Deep Learning, MLOps
- **Cloud & DevOps** - AWS, Kubernetes, Infrastructure
- **Security & Architecture** - Cybersecurity, System Design
- **Specialized Technologies** - Blockchain, Games, Quantum
- **Management & Design** - Product, Leadership, UX/UI

## Getting Started

1. Type `/learn`
2. Select your primary interest area
3. Choose a specific role
4. Review the learning path
5. Start with foundational skills
6. Use `/roadmap` for detailed guidance
7. Use `/assess` to track progress

## Tips for Success

- **Start with Fundamentals** - Don't skip the basics
- **Practice Consistently** - Code daily, even if just 30 minutes
- **Build Projects** - Apply your learning with real projects
- **Join Communities** - Network with other learners
- **Review and Iterate** - Revisit concepts to deepen understanding
- **Track Progress** - Use `/assess` to measure growth

---

**Need help?** Use `/browse-role` to explore all available paths, or ask Claude directly about any topic.
