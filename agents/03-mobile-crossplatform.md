## üìö Core Learning Paths

### 1. iOS Development

**Swift Programming:**
- Basics: variables, functions, optionals, closures
- OOP: classes, structs, protocols, extensions
- Advanced: concurrency (async/await), generics, property wrappers

**UI Development:**
- **UIKit** (Traditional)
  - View controllers, navigation, delegates
  - Auto Layout, IBOutlets
  - Storyboards and XIBs

- **SwiftUI** (Modern, Recommended)
  - Declarative syntax
  - State management (@State, @StateObject)
  - Navigation (NavigationStack, NavigationSplitView)
  - Modifiers and environment

**Architecture Patterns:**
- MVC, MVP, MVVM
- MVVM-C (Coordinator pattern)
- VIPER
- TCA (The Composable Architecture)

**iOS Ecosystem:**
- App lifecycle and delegates
- Networking (URLSession)
- Storage (Core Data, CloudKit)
- Concurrency (async/await, GCD)
- Push notifications
- In-App Purchases
- App Store deployment

### 2. Android Development

**Kotlin Programming:**
- Basics: variables, functions, lambdas, extensions
- OOP: classes, objects, inheritance
- Coroutines for async programming
- Collections and functional programming

**UI Development:**
- **Traditional Views**
  - Layouts (LinearLayout, ConstraintLayout, etc.)
  - RecyclerView and Adapters
  - Fragments and Activities

- **Jetpack Compose** (Modern, Recommended)
  - Composable functions
  - State management
  - Theming and Material Design 3
  - Navigation

**Android Jetpack Libraries:**
- Architecture Components (ViewModel, LiveData)
- Room for local database
- DataStore for preferences
- WorkManager for background tasks
- Hilt for dependency injection

**Architecture Patterns:**
- MVC, MVP, MVVM
- Repository Pattern
- CLEAN architecture
- Single Activity Pattern with Fragments

**Android Ecosystem:**
- Activity & Fragment lifecycle
- Intent and navigation
- Networking (Retrofit, OkHttp)
- Storage (SharedPreferences, Room, Firestore)
- Background processing
- Google Play deployment

### 3. React Native

**JavaScript/TypeScript Foundation:**
- React fundamentals and hooks
- JavaScript async patterns
- Native modules integration

**Core Components:**
- View, Text, TextInput, Image
- FlatList, SectionList
- Navigation systems
- Platform-specific code

**State Management:**
- Redux, Context API
- MobX, Zustand
- Async patterns

**Native Bridge:**
- Platform modules
- Native module creation
- Bridge communication

**Ecosystem:**
- Expo (simplified setup and deployment)
- React Navigation
- NativeBase, React Native Elements (UI libraries)
- Firebase integration
- App Store & Google Play deployment

### 4. Flutter

**Dart Programming:**
- Basics: variables, functions, classes
- Advanced: streams, futures, isolates
- Functional and OOP paradigms

**Flutter Framework:**
- Widgets (Stateless, Stateful, Inherited)
- Material Design and Cupertino (iOS)
- Responsive design
- Animation framework

**State Management:**
- Provider (recommended)
- Riverpod
- BLoC pattern
- Redux, GetX, MobX

**Navigation:**
- Named routes
- GoRouter (modern)
- Deep linking

**Packages & Ecosystem:**
- pub.dev packages
- Firebase integration
- SQLite and local storage
- Networking (http, dio)
- App deployment (iOS and Android)

### 5. Mobile App Features

**Networking:**
- HTTP requests and REST APIs
- GraphQL for mobile
- WebSockets for real-time
- SSL pinning and security

**Storage:**
- Local databases (SQLite, Room, Core Data)
- Key-value storage
- File system management
- Cloud sync (iCloud, Google Drive)

**Background Processing:**
- Background tasks and services
- Work scheduling
- Push notifications
- Periodic tasks

**Testing:**
- Unit testing
- Widget/Component testing
- Integration testing (Espresso, XCUITest)
- E2E testing (Detox, Appium)
