## üìö Core Learning Paths

### 1. SQL & Database Fundamentals

**SQL Mastery:**
- SELECT, INSERT, UPDATE, DELETE operations
- JOINs (INNER, LEFT, RIGHT, FULL, CROSS)
- Aggregations and GROUP BY
- Window functions and CTEs
- Subqueries and nested queries
- Query optimization and EXPLAIN plans
- Transactions and ACID properties

**Relational Databases:**
- PostgreSQL (recommended)
- MySQL, MariaDB
- Microsoft SQL Server
- Oracle Database
- Schema design and normalization

**NoSQL Databases:**
- MongoDB (document-based)
- Cassandra (time-series, wide-column)
- Neo4j (graph databases)
- Redis (in-memory)
- DynamoDB (AWS)

### 2. Data Engineering

**Big Data Technologies:**
- **Apache Spark** (most important)
  - RDDs and DataFrames
  - Spark SQL
  - PySpark and Scala
  - Distributed computing concepts
  - Performance tuning

- **Apache Hadoop**
  - HDFS (Hadoop Distributed File System)
  - MapReduce
  - YARN resource management
  - Hive and Pig

- **Apache Kafka**
  - Event streaming
  - Topics, partitions, brokers
  - Consumer groups
  - Real-time data pipelines

**ETL & Data Pipelines:**
- **Apache Airflow** (recommended for orchestration)
  - DAGs (Directed Acyclic Graphs)
  - Task scheduling and monitoring
  - Operators and sensors
  - Extensibility

- **Luigi**, **Prefect**, **Dagster**
- **dbt** (data build tool) - modern SQL-based transformation

**Cloud Data Platforms:**
- **AWS**: S3, Redshift, EMR, Glue, DMS, Kinesis
- **Google Cloud**: BigQuery, Dataflow, Cloud Storage
- **Azure**: Synapse Analytics, Data Factory, Databricks

**Data Warehousing:**
- Snowflake (cloud-native, recommended)
- BigQuery (fully managed)
- Redshift (AWS)
- Dimensional modeling (star schema, snowflake schema)
- Slowly Changing Dimensions (SCD)

### 3. Data Analysis

**Statistics & Math:**
- Descriptive statistics (mean, median, std dev)
- Probability distributions
- Hypothesis testing
- A/B testing and statistical significance
- Correlation and causation
- Regression analysis

**Data Visualization:**
- **Tools**: Tableau, Power BI, Looker, Grafana
- **Libraries**: Matplotlib, Seaborn, Plotly
- Dashboard design principles
- Interactive visualizations

**Business Metrics:**
- KPIs (Key Performance Indicators)
- Funnel analysis
- Cohort analysis
- Retention and churn analysis
- CAC (Customer Acquisition Cost)
- LTV (Lifetime Value)

### 4. Data Science & ML

**Programming:**
- Python (essential)
  - Pandas (data manipulation)
  - NumPy (numerical computing)
  - Scikit-learn (ML algorithms)
  - TensorFlow, PyTorch (deep learning)
  - Jupyter notebooks
- R (statistical computing)
- SQL (always important)

**Machine Learning:**
- Supervised learning (classification, regression)
- Unsupervised learning (clustering, dimensionality reduction)
- Feature engineering
- Model evaluation and metrics
- Cross-validation
- Hyperparameter tuning

**Data Science Workflow:**
1. Problem definition
2. Data collection and exploration (EDA)
3. Data preprocessing and cleaning
4. Feature engineering
5. Model building and training
6. Model evaluation
7. Deployment and monitoring
8. Iterative improvement

### 5. Technologies & Tools

**Programming Languages:**
- Python (primary)
- SQL (essential)
- Scala (for Spark)
- R (statistical analysis)

**Data Tools:**
- Apache Spark and Hadoop
- Apache Airflow
- dbt (SQL transformation)
- Kafka (streaming)
- pandas, NumPy, Scikit-learn
- Jupyter, RStudio

**Visualization:**
- Tableau (leading BI tool)
- Power BI (Microsoft)
- Looker (Google Cloud)
- Grafana (monitoring)
- Matplotlib, Plotly (Python)

**Cloud Platforms:**
- AWS (S3, Redshift, EMR, Glue)
- Google Cloud (BigQuery, Dataflow)
- Azure (Synapse, Data Factory)

### 6. Modern Data Stack (2025)

- **ELT instead of ETL** (cloud compute is cheap)
- **dbt** for SQL-based transformations
- **Snowflake** or **BigQuery** for data warehouse
- **Kafka** or cloud-native streaming
- **Analytics engineering** combining analytics and engineering
- **Data mesh** architecture for large organizations
- **DataOps** - DevOps practices for data
