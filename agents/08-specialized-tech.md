### 1. Blockchain Development

**Blockchain Fundamentals:**
- Distributed ledger technology
- Consensus mechanisms (PoW, PoS, PoA)
- Cryptographic hashing
- Digital signatures
- Merkle trees
- Smart contracts concept

**Popular Blockchains:**
- **Ethereum** (smart contracts leader)
- **Solana** (high-performance)
- **Polygon** (scaling solution)
- **Arbitrum** (Layer 2)
- **Base** (Coinbase's L2)
- **Binance Smart Chain**
- **Bitcoin** (original, UTXO model)

**Smart Contract Development:**

**Solidity (Ethereum):**
- Language syntax and semantics
- Smart contract structure
- Gas optimization
- Security considerations
- ERC-20, ERC-721 standards
- Upgradeable contracts (proxies)

**Alternative Languages:**
- Rust (Solana)
- Vyper (Ethereum, Python-like)
- Cairo (StarkNet)
- Move (Aptos, Sui)

**Development Tools:**
- **Hardhat** (Ethereum development)
- **Foundry** (Rust-based, fast)
- **Truffle** (Smart contract framework)
- **Remix** (Browser IDE)
- **Anchor** (Solana framework)

**DeFi (Decentralized Finance):**
- Automated Market Makers (AMMs)
- Lending protocols
- Yield farming
- Governance tokens
- Liquidity pools
- Swaps and exchanges

**NFTs & Web3:**
- NFT standards (ERC-721, ERC-1155)
- IPFS (distributed storage)
- Marketplaces
- Collections and metadata
- Royalties and creator tools

**Testing & Auditing:**
- Unit testing with Hardhat/Foundry
- Integration testing
- Security auditing
- Formal verification

### 2. Game Development

**Game Engines:**
- **Unity** (most popular, C#)
  - 2D and 3D development
  - Asset store
  - Community support
  - Mobile and console support

- **Unreal Engine** (AAA quality, C++)
  - Advanced graphics
  - Blueprints visual scripting
  - Performance optimization
  - Learning curve steeper

- **Godot** (open-source, GDScript)
  - Lightweight
  - Growing community
  - Good for 2D games
  - Increasing 3D capabilities

- **Custom Engines** (performance-critical)

**Game Development Fundamentals:**
- Game loops and frame rate
- Input handling
- Physics engines
- Collision detection
- Animation systems
- Audio systems
- Networking (multiplayer)

**Graphics & Rendering:**
- 3D graphics pipeline
- Shaders (vertex, fragment)
- Materials and textures
- Lighting and shadows
- Post-processing effects
- Ray tracing

**Game Design & Mechanics:**
- Game design document (GDD)
- Gameplay mechanics
- Level design
- User interface design
- Player progression
- Monetization (F2P, premium)

**Game Programming Patterns:**
- Model-View-Controller (MVC)
- Entity-Component-System (ECS)
- State machines
- Observer pattern
- Singleton pattern (use cautiously)
- Object pooling

**Tools & Middleware:**
- Wwise (audio)
- FMOD (audio)
- Photon (multiplayer)
- Playfab (backend services)
- Steam (distribution)

**Specialized Areas:**
- **VR/AR Development** - Meta Quest, Apple Vision Pro
- **Mobile Games** - iOS and Android optimization
- **Multiplayer Systems** - Server architecture, matchmaking
- **Procedural Generation** - Terrain, levels, content
